## Quick Start

1. Clone the repository:
    ```bash
    git clone https://github.com/yagyandatta/weblancer.git
    ```
2. Install dependencies:
    ```bash
    cd weblancer
    npm install
    ```
3. Start the development server:
    ```bash
    npm run dev
    ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
